DeepSeek V4 Flash is now available on Ollama's cloud
 
DeepSeek-V4-Flash is now available to run on Ollama's cloud using the latest NVIDIA Blackwell hardware. (DeepSeek-V4-Pro is coming very soon).
 
 
Download Ollama
 
To use DeepSeek-V4-Flash with Claude Code, run:
 
ollama launch claude --model deepseek-v4-flash:cloud
 
For use with OpenClaw:
 
ollama launch openclaw --model deepseek-v4-flash:cloud
 
For use with Hermes Agent:
 
ollama launch hermes --model deepseek-v4-flash:cloud
 
Chat with the model:
 
ollama run deepseek-v4-flash:cloud
 
See the model page for more information and integrations.
 
 
 
Please bear with us as we continue to add GPU capacity to Ollama's Cloud.
