Ollama 0.24 is now available with support for OpenAI’s Codex App
Ollama 0.24 is now available with support for the Codex App, OpenAI's desktop experience for working on Codex threads in parallel with built-in worktree support and git functionality.
 
 

Get started
Download Ollama 0.24, then run:
 
 
ollama launch codex-app
 

Built-in Browser
Codex can load local servers and sites in its built-in browser, enabling you to directly annotate on the page to request changes.
 
 

Review Mode
Review code inside the app, leave comments, and iterate without leaving your workspace.
 
 

Choosing a Model
For coding and agent tasks:
  • kimi-k2.6:cloud (with vision support)
  • glm-5.1:cloud
 
If you don't yet have a paid subscription with Ollama's cloud, choose a model that supports reliable tool calling:
  • nemotron-3-super:cloud
  • gemma4:31b:cloud
  • qwen3.6
 

Restoring default models
To restore the default Codex profile, use --restore:
 
 
ollama launch codex-app --restore
