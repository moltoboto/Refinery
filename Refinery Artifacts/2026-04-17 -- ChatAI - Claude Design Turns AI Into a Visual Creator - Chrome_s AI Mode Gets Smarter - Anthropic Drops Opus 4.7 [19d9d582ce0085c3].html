Claude Design: Anthropic's New AI Tool Creates Slides, Prototypes, and More

Anthropic launched Claude Design, an experimental feature that lets users generate visual assets — prototypes, presentations, and one-pagers — directly inside Claude using natural language. Powered by Claude Opus 4.7, the tool targets non-designers like founders and product managers who need to quickly turn ideas into polished visuals without traditional design software. Users can export as PDF, PPTX, or shareable links, and continue editing in Canva. The launch follows Anthropic CPO Mike Krieger's departure from Figma's board — days before the competing product was announced. Read more →

Chrome's AI Mode Just Got Smarter With Side-by-Side Search and Tab Integration

Google expanded Chrome's AI Mode with a split-screen layout that opens webpages alongside AI responses, letting users compare sources and ask follow-up questions without leaving the page. Chrome can now also search across multiple open tabs — users can select tabs, images, or PDFs and include them as context in AI queries. A new menu provides quick access to tools like Canvas and image generation directly within the browser. Rolling out in the U.S. first. Read more →

Claude Opus 4.7 Now Available With Improved Coding and Vision Features

Anthropic released Claude Opus 4.7, an updated flagship model focused on advanced software engineering, now available across the API and major cloud platforms including Amazon Bedrock, Google Cloud Vertex AI, and Microsoft Foundry. The model delivers stronger consistency and instruction-following in multi-step coding tasks, supports higher-resolution image inputs (up to 2,576 pixels), and introduces new cybersecurity safeguards that will inform future releases. A new "xhigh" effort setting lets developers balance reasoning depth and latency. Read more →

Google Rolls Out Gemini Desktop App and New AI Search Experience

Google launched a native Gemini app for macOS and a Windows search application, moving key AI services beyond the browser. The macOS app provides the full Gemini experience — including Deep Research, Canvas, and image/video generation — via a keyboard shortcut. The Windows app lets users search the web and local files through a floating interface with AI Overviews. Both apps can pull context from open windows. Read more →

OpenAI Expands Agents SDK With Sandbox and New Tools for Enterprise AI

OpenAI updated its Agents SDK with sandboxed environments that let AI agents operate in isolated workspaces, limiting access to specific files and systems. A new in-distribution harness enables agents to interact with approved tools inside a defined workspace, aimed at building "long-horizon" agents that complete extended, multi-step tasks. Available in Python now, with TypeScript support planned. Read more →

India's Vibe-Coding Startup Emergent Enters OpenClaw-Like AI Agent Space

Emergent, the Indian startup behind a vibe-coding platform with 8 million builders, launched Wingman — a messaging-first autonomous AI agent that operates through WhatsApp, Telegram, and iMessage. Wingman can handle routine tasks across email, calendars, and workplace tools autonomously, while seeking user approval for more consequential actions. The company raised $70M in January at a $300M valuation from SoftBank and Khosla Ventures. Read more →

Image courtesy of Zoom

Zoom Partners With Tools for Humanity to Verify Humans in Video Calls

Zoom announced a partnership with Tools for Humanity to integrate World ID Deep Face into Zoom Meetings, enabling real-time verification that participants are human. Enrolled users verify via World App before joining, and a "Verified Human" badge appears on their video tile. No personal data is shared with Zoom or other participants. The integration targets high-stakes scenarios in financial services, healthcare, and executive communications — where deepfake impersonation risk is growing. Beta access is opening later this year. Read more →

OpenAI's Codex Gets Desktop Control, Memory, and 90+ New Plugins

OpenAI released a major Codex update giving it the ability to operate your desktop — seeing, clicking, and typing with its own cursor while multiple agents run in parallel. The update adds persistent memory so Codex retains context across sessions, image generation via gpt-image-1.5, an in-app browser for frontend iteration, and 90+ new plugins including Atlassian, CircleCI, GitLab, and Microsoft Suite. Codex can now schedule future work for itself and wake up automatically to continue long-term tasks across days or weeks. Read more →

🎮 GAMING

Image courtesy of Roblox

Roblox Upgrades AI Assistant With Planning Mode and 3D Generation Features

Roblox launched an upgraded AI Assistant in Roblox Studio with a new "Planning Mode" that breaks game development into planning, feedback, and execution steps. Creators describe a game idea in plain language and the Assistant produces a structured, editable plan before any code or assets are generated. New Mesh Generation creates fully textured 3D objects directly in the game world. The Assistant can also run playtests, capture screenshots, simulate inputs, and propose fixes automatically. Roblox is working on support for multiple parallel agents and integration with Claude, Cursor, and Codex. Read more →

UK Launches $675M Sovereign AI Fund to Cut Foreign Tech Reliance

The UK government launched a $675 million sovereign AI fund exclusively for homegrown startups, marking its most aggressive push to reduce dependence on U.S. and Chinese AI technology. The fund targets British-based companies developing large language models, computer vision, and enterprise AI — with recipients likely required to keep technology and data within UK borders. It follows similar moves by France (€500M) and Germany (€1B), as European governments increasingly treat AI development as critical national infrastructure. Read more →

Luma launched Innovative Dreams, a new production company with Wonder Project, to bring AI tools into film and TV. Their first project, "The Old Stories: Moses" starring Ben Kingsley, will stream on Amazon Prime Video this spring. Canva's AI 2.0 Turns Text Prompts Into Full Design Workflows

Canva unveiled AI 2.0, a major update that adds a conversational interface across its 200-million-user platform. Instead of navigating menus and templates, users can issue text prompts to generate designs, modify layouts, and apply brand elements — with the AI orchestrating changes across multiple tools in real time. The update targets teams handling marketing and content production by simplifying repetitive tasks like generating variations or adapting assets across formats. Paid subscribers get early access. 💰 FINANCE

InsightFinder Raises $15M to Diagnose Why AI Agents Fail in Production

AI observability startup InsightFinder secured $15 million in Series B funding, led by Yu Galaxy, to help enterprises pinpoint where and why their AI agents fail in production. Unlike traditional monitoring tools, InsightFinder analyzes the data, model, and infrastructure as a unified system. The platform caught a fraud detection model drifting at a major credit card company — tracing the root cause to outdated cache on specific servers, not the model itself. Clients include UBS, NBCUniversal, Lenovo, Dell, and Google Cloud. Read more →

Antioch Raises $8.5M to Build the "Cursor for Robotics" Simulation Platform

Antioch, founded by former Tesla Autopilot and Google DeepMind engineers, raised $8.5 million to build a cloud simulation platform that lets robotics teams create digital twins and run thousands of tests in parallel — replacing expensive physical test environments. The round was led by A* and Category Ventures, with angel backing from Palantir's Shyam Sankar. Co-founder Harry Mellsop compares the platform to Cursor for autonomous systems: always giving teams access to the latest simulation and AI tools. Read more →

Dairy Queen Deploys AI Chatbots in Drive-Thrus to Boost Speed and Sales

Dairy Queen is rolling out AI-powered voice ordering systems at drive-thrus across the U.S. and Canada, expanding a pilot with Presto that automates order-taking and introduces upselling during transactions. The system suggests add-ons and upgrades to increase average order value during peak hours. Questions remain about how automated the systems truly are — a prior Bloomberg investigation found some AI drive-thru systems rely on human workers when orders get complex. Read more →

AI Traffic to US Retailers Rose 393% in Q1 — and It's Boosting Revenue Too

AI-driven traffic to U.S. retail websites rose 393% year-over-year in Q1 2026, according to Adobe Analytics data covering 1 trillion+ visits. AI visitors now convert 42% better than human shoppers — a complete reversal from March 2025, when AI traffic converted 38% worse. Revenue per visit from AI sources is 37% higher, engagement rates are 12% higher, and shoppers spend 48% longer on sites. However, roughly a quarter of retailer homepage content still isn't optimized for LLMs. Read more →
